# 3D Designing


Yesterday I learned how to design something on the Onshape website. First, I watched some YouTube tutorials to understand how to use Onshape and learn the basic tools. After watching the tutorials, I tried designing a bottle on Onshape with the help of my friends. The video helped me understand the steps needed to create a simple 3D model.

This is the video I watched to learn the process:
https://youtu.be/iDgm6AVrNjg?si=9k9Y3nd7j8qs3DxN

During this activity, I learned how to use different tools in Onshape such as Sketch, Revolve, and Fillet. These tools help create shapes and convert them into a 3D design.



## Designing Bottle on Onshape

Firstly, I started by drawing the shape of the bottle from one side using the Sketch tool. I created a 2D outline of half of the bottle. This outline is important because it will later be used to create the full 3D shape of the bottle.

After finishing the sketch of the bottle shape, I used the Revolve tool. This tool rotates the sketch around a center axis to create a complete 3D object. By revolving the half shape I drew earlier, the bottle became a full 3D model.

Next, I used Fillet 2 (Sketch tool) to add smooth curves and small design lines on the cap and neck of the bottle. This made the bottle look more realistic and improved the overall design.
