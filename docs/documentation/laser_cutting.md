# Laser Cutting
Today I learned how to use a laser cutting machine. The lesson was taught by Yangtshel, and I found it very easy to understand because the steps were explained clearly. At first, I thought laser cutting would be difficult, but after learning the process I realized that most of the work is done on the computer before using the machine. The main tasks include creating or editing a design, setting the correct parameters, and preparing the machine properly.
Laser cutting is a digital fabrication process where a high-powered laser beam is used to cut or engrave materials such as wood, acrylic, cardboard, or leather. The laser follows the design created on the computer and cuts the material very precisely.

## Designing the File
We used the Inkscape software to create the design that we wanted to cut. Inkscape is a vector graphics editor that allows us to draw shapes, lines, and patterns. These vector designs can then be sent to the laser cutting machine.
While creating the design in Inkscape, we have to make sure that:
The lines are vector lines, not images.
The lines are thin strokes so the laser can follow them properly.
The design fits within the size of the material we are using.
After finishing the design, the file is prepared so it can be sent to the laser cutter.

## Preparing the Laser Cutting Machine
Before starting the cutting process, we have to prepare the machine and make sure everything is ready. This includes turning on several systems that help the machine work safely and efficiently.
First, we turn on:
The laser cutting machine
The air compressor
The exhaust system

The air compressor helps blow air onto the cutting area to prevent burning and improve cutting quality. The exhaust system removes smoke and fumes produced when the laser cuts the material.
After turning on the systems, we adjust the laser head height by using the auto-focus function. This step is important because the correct distance between the laser head and the material ensures accurate and clean cutting.
Next, we set the machine parameters such as:
Power – controls how strong the laser beam is
Speed – determines how fast the laser moves across the material
Frequency – controls how often the laser pulses during cutting
These settings depend on the type and thickness of the material being used. If the power is too high or the speed is too slow, the material might burn. If the power is too low, the laser might not cut through the material completely.

Process
Firstly, I went to the Pinterest website to select a picture that I wanted to print using the laser cutting machine. Pinterest has many different images and design ideas, so it is easy to find a suitable picture for laser cutting.
After choosing the image, I copied the picture and pasted it into **Inkscape. Inkscape is a vector graphics software that allows us to edit images and convert them into a format that the laser cutter can understand.

Next, I clicked on the "Trace Bitmap" option in Inkscape. This tool converts a normal image (raster image) into a vector outline. Vector lines are important because the laser cutting machine follows these lines to cut or engrave the material. After using Trace Bitmap, the image becomes a clean outline that the machine can read.
After converting the image into vector lines, the design is ready to be sent to the laser cutting machine.
Final Result
After preparing the machine and sending the design to the laser cutter, the machine follows the vector lines and cuts the material based on the design. The process is very precise and produces clean edges.

Safety While Using the Laser Cutter
Safety is very important when using a laser cutting machine. Some important safety rules include:
Always check the material before cutting.
Never leave the machine unattended while it is running.
Make sure the exhaust system is turned on to remove smoke.
Keep the machine clean and free of flammable materials.
Following these safety steps helps prevent accidents and ensures the machine works properly.
